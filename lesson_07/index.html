Масиви

Масиви: Теорія

Масив — це

Масив — це спеціальний тип змінної, який дозволяє зберігати багато значень одночасно в одному місці.

Наприклад, нам треба зберегти відповіді до 5-ти кроків гри. Замість того, щоб створювати 5 окремих змінних для кожної відповіді, можемо створити один масив: 

// 5 окремих змінних:
let answer1 = 1;
let answer2 = 1;
let answer3 = 2;
let answer4 = 2;
let answer5 = 1;
// Один масив:
let answers = [1, 1, 2, 2, 1];

Доступ до елементів масиву (індекси)

Кожен елемент у масиві має своє місце (індекс). Індекси починаються з 0. 

let fruits = ["apple", "banana", "orange"];

console.log(fruits); // ['apple', 'banana', 'orange']
console.log(fruits[0]); // 'apple'
console.log(fruits[1]); // 'banana'
console.log(fruits[2]); // 'orange'

fruits[1] = "kiwi";
console.log(fruits); // ['apple', 'kiwi', 'orange']

Довжина масиву (.length)

Ми можемо дізнатися, скільки елементів у масиві, використовуючи властивість .length. 

console.log(answers.length); // 5
console.log(fruits.length); // 3

push/pop: Теорія

push()

Метод push() використовується, щоб додати новий елемент в кінець масиву: 

let fruits = ["apple", "banana"];
fruits.push("orange");
console.log(fruits); // ['apple', 'banana', 'orange']

pop()

Метод pop() використовується, щоб видалити останній елемент масиву: 

let fruits = ["apple", "banana", "orange"];
fruits.pop();
console.log(fruits); // ['apple', 'banana']

forEach: Теорія

forEach()

Ми можемо пройтися по кожному елементу масиву за допомогою методу forEach(), що дозволяє виконати певну дію для кожного з цих елементів. 

let fruits = ["apple", "banana", "orange", "peach", "mango"];
// без методу forEach():
console.log(fruits[0]);
console.log(fruits[1]);
console.log(fruits[2]);
console.log(fruits[3]);
console.log(fruits[4]);
// з методом forEach() (варіант 1):
fruits.forEach(function(fruit) {
  console.log(fruit);
});
// з методом forEach() (варіант 2):
fruits.forEach((fruit) => {
  console.log(fruit);
});

.indexOf()

Метод .indexOf() допомагає знайти індекс певного елемента в масиві. Якщо елемент не знайдено, повертається -1. 

let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("banana");
console.log(index); // 1

numbers = [10, 100, 30, 40, 50, 7]

  1. Знайти добуток (product) всіх чисел масиву numbers.
  2. Знайти найменше (min) число в масиві.

Приклад розв'язання

// добуток
let product = 1;
numbers.forEach((num) => {
  product *= num;
});
console.log("добуток: " + product);

// найменше число
let min = numbers[0];
numbers.forEach((num) => {
  if (num < min) {
    min = num;
  }
});
console.log("найменше число: " + min);

// індекс найменшого числа
console.log("індекс найменшого числа: " + numbers.indexOf(min));

splice: Теорія

Splice()

З допомогою методу splice() можна:

  • видалити з масиву один елемент з певним індексом;
  • видалити з масиву кілька елементів починаючи з певного індексу;
  • видалити з масиву один або більше елементів і вставити на їхнє місце якісь інші елементи. 
let box = ["red", "green", "yellow", "blue", "white"];

// знаходимо індекс 3, видаляємо 1 елемент:
box.splice(3, 1);

// знаходимо індекс 2, видаляємо 3 елементи:
box.splice(2, 3); 

// знаходимо індекс 1, видаляємо 1 елемент
// додаємо на його місці 1 новий елемент:
box.splice(1, 1, "black"); 

// знаходимо індекс 1, видаляємо 1 елемент
// додаємо на його місці 2 нові елементи:
box.splice(1, 1, "black", "orange");
